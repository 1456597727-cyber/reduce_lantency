# 直播观看服务 - 服务器部署指南

## 1. 买服务器

推荐：腾讯云轻量应用服务器
- 配置：1核2G（最便宜款 ~40元/月）
- 系统：Ubuntu 22.04 或 CentOS 7
- 买完后你会得到：IP地址 + root密码

## 2. 连上服务器

在你的 Windows 电脑上：

```bash
# 用 SSH 工具连接（Windows 自带或下载 Xshell/PuTTY）
ssh root@你的服务器IP
# 输入密码
```

## 3. 安装环境

```bash
# 更新系统
apt update && apt upgrade -y  # Ubuntu
# 或 yum update -y  # CentOS

# 安装 Python 3 和 pip
apt install python3 python3-pip -y

# 安装防火墙（如果还没装）
apt install ufw -y
ufw allow 22    # SSH
ufw allow 5001  # 你的网站端口
ufw enable
```

## 4. 上传代码

在你 Windows 电脑上，把 `D:\stream_service` 整个文件夹传到服务器：

```bash
# 在 Windows 上用 scp（PowerShell 或 cmd）
scp -r D:\stream_service root@你的服务器IP:/root/stream_service
```

## 5. 在服务器上启动

```bash
cd /root/stream_service

# 安装 Python 依赖
pip3 install -r requirements.txt

# 后台启动（关了SSH也不会停）
nohup python3 app.py > server.log 2>&1 &

# 查看日志
tail -f server.log
```

## 6. 访问

浏览器打开：`http://你的服务器IP:5001`

---

## 日常管理

```bash
# 查看是否在运行
ps aux | grep app.py

# 停止服务
pkill -f app.py

# 重新启动
cd /root/stream_service && nohup python3 app.py > server.log 2>&1 &

# 修改管理员密码（首次必做！）
# 编辑 app.py，搜索 admin123 改掉
# 或者用环境变量: export ADMIN_PASS=你的密码
```

## 域名（可选）

买个域名（~50元/年），解析到服务器IP，就不用记IP了。

在服务器上安装 nginx 做反向代理，实现：
- 80端口 → 5001端口
- HTTPS 加密
